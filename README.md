"# DynamicKafkaConsumer" 

for java8 , use branch java8

This application support to 
* create consumer
* check active consumers
* activate consumer
* deactivate consumer
* pause consumer
* resume consumer


Note : 
Need cron job for cleaning data automatically for data log
